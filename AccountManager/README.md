# Purpose
Simple sign-up/log-in application.
# Technologies practiced:
### Express
### Express-session
### Bcrypt
### Mongoose/MongoDB
### Cookies

## To get running
### Install dependencies
```
npm install express
npm install mongoose
npm install bcrypt
```
### Create DB
```
mongo
use accountManagerApp
exit
```
